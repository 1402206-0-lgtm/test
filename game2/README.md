# 🐍 贪吃蛇游戏

这是一个使用HTML5、CSS3和JavaScript创建的贪吃蛇游戏，专为Visual Studio Code环境设计。

## 🚀 如何运行

1. 在VS Code中打开项目文件夹
2. 安装"Live Server"扩展（如果尚未安装）：
   - 按 `Ctrl+Shift+X` 打开扩展面板
   - 搜索 "Live Server"
   - 点击安装
3. 右键点击 `index.html` 文件
4. 选择 "Open with Live Server"
5. 游戏将在默认浏览器中打开

## 🎯 游戏功能

- ✅ 经典的贪吃蛇游戏玩法
- ✅ 多种食物类型：普通、奖励、加速
- ✅ 分数、长度、速度统计
- ✅ 最高分记录（本地存储）
- ✅ 多种游戏主题
- ✅ 游戏设置：网格大小、速度、穿墙模式
- ✅ 移动端触摸控制
- ✅ 响应式设计

## 🕹️ 控制说明

### 键盘控制：
- **W/A/S/D 或方向键**：控制蛇移动
- **空格键**：加速移动
- **P键**：暂停/继续游戏
- **R键**：重新开始游戏
- **+/-键**：调整游戏速度
- **F11**：全屏模式

### 鼠标/触摸控制：
- 点击屏幕上的方向按钮（移动端）
- 使用设置面板调整游戏参数

## 🎨 游戏特色

1. **多种食物**：
   - 🟥 普通食物：+10分
   - 🟧 奖励食物：+30分
   - 🟦 加速食物：提高游戏速度

2. **主题切换**：
   - 经典绿
   - 暗黑紫
   - 海洋蓝
   - 霓虹效果

3. **游戏设置**：
   - 网格大小：20×20 / 25×25 / 30×30
   - 游戏速度：1-10级可调
   - 穿墙模式：开启/关闭
   - 显示网格：开启/关闭

4. **本地存储**：
   - 最高分记录
   - 最长蛇长度
   - 游戏次数统计

## 📱 响应式设计

游戏支持各种屏幕尺寸：
- 桌面电脑：完整界面，键盘控制
- 平板电脑：自适应布局，触摸控制
- 手机：垂直堆叠布局，触摸按钮

## 🎮 游戏规则

1. 控制蛇吃到食物（红色方块）
2. 每吃一个食物：
   - 蛇的长度增加1
   - 分数增加（根据食物类型）
3. 游戏结束条件：
   - 撞到墙壁（穿墙模式关闭时）
   - 撞到自己身体
4. 游戏目标：
   - 获得尽可能高的分数
   - 让蛇尽可能长

## 🐛 故障排除

如果游戏无法运行：

1. 确保已安装Live Server扩展
2. 检查浏览器控制台是否有错误（F12）
3. 确保所有文件都在同一目录下
4. 尝试使用Chrome或Firefox浏览器
5. 清除浏览器缓存后重试
