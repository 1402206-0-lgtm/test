# 🎮 俄罗斯方块游戏

这是一个使用HTML5、CSS3和JavaScript创建的俄罗斯方块游戏，专为Visual Studio Code环境设计。

## 🚀 如何运行

1. 在VS Code中打开项目文件夹
2. 安装"Live Server"扩展（如果尚未安装）：
   - 按 `Ctrl+Shift+X` 打开扩展面板
   - 搜索 "Live Server"
   - 点击安装
3. 右键点击 `index.html` 文件
4. 选择 "Open with Live Server"
5. 游戏将在默认浏览器中打开

## 🎯 游戏功能

- ✅ 完整的俄罗斯方块游戏逻辑
- ✅ 7种经典方块形状
- ✅ 分数、等级和行数统计
- ✅ 下一个方块预览
- ✅ 游戏暂停和重新开始
- ✅ 响应式设计
- ✅ 美观的UI界面

## 🕹️ 控制说明

- **左右箭头** ← → : 移动方块
- **上箭头** ↑ : 旋转方块
- **下箭头** ↓ : 加速下落
- **空格键** : 直接落到底部
- **P键** : 暂停/继续游戏
- **R键** : 重新开始游戏

## 📱 响应式设计

游戏支持各种屏幕尺寸：
- 桌面电脑：完整界面
- 平板电脑：自适应布局
- 手机：垂直堆叠布局

## 🎨 自定义选项

你可以通过修改以下文件来自定义游戏：

1. `game.js`:
   - 修改 `COLORS` 数组改变方块颜色
   - 修改 `SHAPES` 数组改变方块形状
   - 调整 `dropInterval` 改变游戏速度

2. `style.css`:
   - 修改颜色主题
   - 调整布局和间距
   - 自定义按钮样式

## 🐛 故障排除

如果游戏无法运行：

1. 确保已安装Live Server扩展
2. 检查浏览器控制台是否有错误（F12）
3. 确保所有文件都在同一目录下
4. 尝试使用Chrome或Firefox浏览器
